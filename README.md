# peng-poi
基于POI封装的支持大数据量的Excel导入导出库，包含实际项目中常用的功能封装。

##### 安装：

maven:

```
<dependency>
    <groupId>com.github.zzlhy</groupId>
    <artifactId>peng-poi</artifactId>
    <version>1.0</version>
</dependency>

```

Gradle:

```
compile group: 'com.github.zzlhy', name: 'peng-poi', version: '1.0'
```
